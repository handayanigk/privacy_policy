Kebijakan Privasi Aplikasi e-Collect

Terakhir diperbarui: 25 Juni 2025

Aplikasi e-Collect dikembangkan oleh PDAM GK sebagai alat bantu internal untuk mendukung efisiensi kegiatan penagihan lapangan. Kami menghargai privasi pengguna dan berkomitmen untuk melindungi data pelanggan serta pengguna aplikasi.

1. Pengumpulan Data Pribadi

Aplikasi e-Collect tidak mengumpulkan atau menyimpan data pribadi pengguna aplikasi. Seluruh data yang ditampilkan (seperti histori tagihan, lokasi pelanggan, atau status penagihan) hanya ditarik dari sistem backend PDAM GK dan tidak disimpan secara lokal maupun dikirim ke pihak ketiga.

2. Akses Lokasi

Aplikasi e-Collect dapat menggunakan akses lokasi perangkat secara temporer, hanya saat diperlukan untuk mencatat posisi kunjungan petugas penagihan. Lokasi tersebut:

  • Tidak disimpan di perangkat pengguna,
  • Tidak digunakan untuk pelacakan pengguna,
  • Tidak dikaitkan dengan identitas pribadi petugas.

3. Penggunaan Data Pelanggan

Data yang tampil di aplikasi (seperti nama pelanggan, nomor sambungan, histori tagihan, dan status pembayaran) sepenuhnya berasal dari sistem informasi internal PDAM GK dan digunakan hanya untuk keperluan operasional penagihan. Aplikasi tidak menyimpan atau memproses data pelanggan secara mandiri.

4. Keamanan

PDAM GK menjamin bahwa seluruh komunikasi data antara aplikasi dan server dilakukan secara aman melalui jalur terenkripsi. Tidak ada informasi pelanggan yang disimpan secara permanen di perangkat pengguna.

5. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini sewaktu-waktu. Perubahan akan diinformasikan melalui pembaruan aplikasi atau kanal resmi PDAM GK.
